Descrição

MoodPrism é um sistema distribuido para a análise e motorização de fadiga mental dos utilizadores. É composto por vários clientes e um repositório. OS clientes obtêm informação retirada de vários sensores, como teclado e rato, que é posteriormente enviado para o repositório. O repositório analisa esta informação para criar valores estatísticos e padrões relativos ao estado de espirito e da fadiga mental dos utilizadores. Esta troca de mensagens será levada a cabo com recurso a um broker, como o Kafka. Para simular o cenário de utilização, será utilizado o hostpad. A informaçãoo será disponiblizada atravéz de uma aplicação web que será direcionada, por exemplo, a um administrador de sistemas.
